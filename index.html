Recent Projects

The PokeGear Radio project is a music player with a Pokemon theme that plays songs from different regions in the main pokemon games based on the position of the selector. The project consists of an HTML file that defines the structure of the page, a CSS file for styling, a SongPlayer.js file for setting up the audio player and handling song selection and playback, and a Selector.js file that implements a draggable selector on a globe graphic to change the region from which songs are played. Users will be able to listen to Pokemon music, control playback using buttons, and drag the globe selector to play songs from different regions. The music player will randomly select songs to play, and users can loop songs, skip to the next song, and adjust the volume. The clock will also update in real-time.
